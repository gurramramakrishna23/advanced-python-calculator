
# Advanced Python Calculator

An advanced menu-driven calculator built using Python and executed in Google Colab.

## Features
- Addition, Subtraction, Multiplication, Division
- Power and Modulus operations
- Square Root using math library
- Input validation and error handling

## Technologies Used
- Python
- Google Colab

## How to Run
1. Open the notebook file
2. Run all cells
3. Follow the menu instructions
